# AI_Incident_Root_Cause_Analyser


Modern enterprise applications generate massive volumes of logs, alerts, metrics, and operational events
across distributed systems. During production incidents, identifying the actual root cause becomes
time-consuming and impacts business continuity.
Build an intelligent platform that helps engineering and operations teams rapidly identify, analyze, and
summarize production incidents using agentic AI workflows powered by LangChain, LangGraph, and
modern GenAI orchestration frameworks.

Kaggle Dataset Used - https://www.kaggle.com/datasets/shubhampatil1999/synthetic-log-data-of-distributed-system
